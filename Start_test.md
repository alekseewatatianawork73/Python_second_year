# Задачи для повторения
-------------------------------------------------------------
1. Пользователь вводит два предложения с клавиатуры, каждое на новой строке. Вам необходимо определить, являются ли эти предложения анаграммами. При этом знаки препинания, пробелы и регистр символов не следует учитывать. (Одно предложение является анаграммой второго, если при перестановке букв в нём мы можем получить второе предложение). Попробуйте решить данную задачу несколькими способами: через словари, через списки, в три строки с помощью списочных выражений.
